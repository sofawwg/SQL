## Task - 1
```sql
SELECT u.name, r.data, r.total_sum, s.servises 
FROM users u
JOIN records r ON u.id = r.user_id AND u.id = 1
JOIN service_records sr ON r.id = sr.record_id
JOIN services s ON sr.service_id = s.id
WHERE r.user_id = 1;
```

## Task - 3
```sql
SELECT u.name, SUM(r.total_sum) AS sum, GROUP_CONCAT(s.services) AS services
FROM users u
JOIN records r ON u.id = r.user_id 
JOIN service_records sr ON r.id = sr.record_id 
JOIN services s ON sr.service_id = s.id
WHERE u.id = 1
GROUP BY u.name;

```
